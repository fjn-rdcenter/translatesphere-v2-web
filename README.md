# TranslateSphere

TranslateSphere is a professional translation platform designed to provide premium document translation services with custom glossaries and a seamless workflow.

## Features

- **Professional Translation**: High-quality document translation.
- **Custom Glossaries**: Manage and use custom glossaries for consistent terminology.
- **Seamless Workflow**: Streamlined process for uploading, translating, and retrieving documents.
- **Modern UI**: Built with a focus on user experience and aesthetics.

## Technology Stack

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)

## Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- npm, pnpm, or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/fjn-rdcenter/translatesphere-v2-web.git
    cd translatesphere-v2-web
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    pnpm install
    # or
    yarn install
    ```

### Running the Development Server

Start the development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To build the application for production:

```bash
npm run build
# or
pnpm build
# or
yarn build
```
